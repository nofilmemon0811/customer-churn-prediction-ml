# 🏦 Customer Churn Prediction (Machine Learning Project)

## 📌 Objective
This project aims to predict whether a bank customer will leave (churn) or stay using machine learning techniques.

---

## 📊 Dataset
Churn Modelling Dataset containing customer details such as:
- Credit Score
- Geography
- Gender
- Age
- Balance
- Number of Products
- Active Membership Status
- Estimated Salary

---

## ⚙️ Steps Performed
- Data Cleaning & Preprocessing
- Encoding Categorical Variables
- Train-Test Split
- Model Training (Logistic Regression & Random Forest)
- Model Evaluation using:
  - Accuracy
  - Precision
  - Recall
  - F1 Score
  - Confusion Matrix
- Feature Importance Analysis

---

## 🤖 Models Used
- Logistic Regression
- Random Forest Classifier

---

## 📈 Key Insights
- Inactive customers are more likely to churn
- Age plays a significant role in churn behavior
- Geography impacts customer retention
- Customers with fewer products tend to leave more

---

## 🧪 Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score

---

## 🚀 Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

---

## 📌 Conclusion
This model helps identify customers at risk of leaving, allowing banks to take preventive actions.
